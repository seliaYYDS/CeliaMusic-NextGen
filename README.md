# Celia Music NextGen

Celia Music NextGen 是一个桌面音乐播放器，重点放在本地音乐管理、播放体验和可定制界面上。它适合希望把本地曲库、歌词显示、桌面组件和在线能力放在同一个应用里统一使用的用户。

## 功能

- 本地音乐导入、扫描与播放
- 歌词显示、沉浸式播放界面与桌面歌词
- 独立灵动岛组件、壁纸模式等桌面扩展能力
- 网易云在线源、歌单与账号相关功能
- 播放状态记忆、缓存、过渡播放与快捷键支持

## 技术栈

- 前端：React、TypeScript、Vite
- 桌面框架：Tauri
- 后端：Rust
- 在线服务：Node.js、本地 NeteaseCloudMusicApi
